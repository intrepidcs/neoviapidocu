# Message Functions Overview - neoVI API

| Name                                                                                  | Description                                                                                |
| ------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| [GetMessages](getmessages-method-intrepidcs-api.md)                                   | Reads messages from the neoVI device.                                                      |
